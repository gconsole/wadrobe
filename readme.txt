Installation

1. Create new database/table using wadrobe.sql file
2. Ensure you have perl installed and at least present in this dir /usr/bin/perl
3. Ensure you have mysql install, with user 'root' and password '' (or you can change it manually in Schema.pm file)
4. Ensure you have all of these cpan modules installed
- Try::Tiny
- Data::Dumper
- Exception::Class
- Parse::CSV
- DBIx::Class::Schema

Run

Go to <localhost>/index.pl
